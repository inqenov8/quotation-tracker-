# VMU Quotation Tracker

A web application to manage custom quotations from intake to approval and dispatch.

## Setup

1. Install dependencies: `npm install`
2. Create a Supabase project and run `schema.sql`.
3. Copy `.env.example` to `.env` and add your Supabase URL and anon key.
4. Run `npm run dev` for development.
5. Deploy to Vercel with environment variables.

See full documentation in the code.
